	SNAPCAT - Super Fast Tree & Cat Utility

	# Overview:
	SnapCat is a lightning-fast command-line utility for generating file trees
	and reading file metadata. Built with TypeScript for maximum performance
	and reliability.

  -------------------------------------------------------------------------------

	# Key Features:

	• Blazing fast file tree generation
	• Smart ignore pattern support (.gitignore, .snapcatignore)
	• Multiple output formats (JSON, Markdown)
	• File metadata including SHA256 hashes
	• Pattern matching with glob support
	• Configurable recursion depth
	• Cross-platform compatibility

  -------------------------------------------------------------------------------

	# Quick Start:

	1. Install: npm install -g snapcat-ts
	2. Tree: snapcat tree [path] [options]
	3. Cat: snapcat cat <patterns> [options]
